PROGRAM 11

Given a 0-indexed integer array nums, find a peak element, and return its index. If the array contains multiple peaks, return
the index to any of the peaks.

Example 1:
Input: nums = [1,2,3,1]
Output: 2

Program:

public class prb11 {
   
    public static int peek(int[] a) {
        int n = a.length;
        int max = 0;
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                if (a[i] > a[j]) {
                    max = i;
                }
            }
        }
        return max;
    }
}
